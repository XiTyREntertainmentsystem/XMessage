# 📱 XiTyR Messenger v3.2

**P2P-мессенджер** с поддержкой WebSocket-сервера, прямым WebRTC-соединением и автоматической ретрансляцией сообщений через сервер (если прямое соединение недоступно).

---

## 🚀 Возможности

- ✅ **Текстовые сообщения** — отправка и получение в реальном времени
- ✅ **Файлы** — передача любых файлов (фото, видео, документы)
- ✅ **Контакты** — добавление, удаление, сохранение в localStorage
- ✅ **Офлайн-сообщения** — сохраняются и доставляются при появлении соединения
- ✅ **Пинг-тест** — команда `/ping` для проверки связи
- ✅ **WebSocket-сервер** — для сигналинга и ретрансляции
- ✅ **Прямое WebRTC** — быстрое соединение в локальной сети
- ✅ **Автоматическая ретрансляция** — если WebRTC не работает, сообщения идут через сервер
- ✅ **Тёмная/светлая тема**
- ✅ **Адаптивный интерфейс** — работает на ПК, телефонах, планшетах
